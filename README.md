# Personal Portfolio Website

This is a responsive personal portfolio website created as a project for the Human and Computer Interaction (HCI) course. It serves as a mock digital resume designed to demonstrate a user interface for showcasing web design skills, technical specialties, and professional services

## Set-up Locally

1. Clone or download this repository to your local computer.
2. Navigate into the project folder.
3. Open `personalportofolio_LatashaIvyKhoe.html` directly in a web browser.
